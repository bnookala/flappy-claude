# Flappy Claude

A terminal-based Flappy Bird game that integrates with Claude Code hooks to provide entertainment during wait times.

## Quick Start

```bash
uvx flappy-claude
```

## Controls

- `SPACE` - Flap
- `Q` - Quit

## Development

```bash
# Run locally
uv run python -m flappy_claude

# Run tests
uv run pytest

# Single-life mode
uv run python -m flappy_claude --single-life
```
